# 📱 Teen Social Media & Mental Health Analysis

An exploratory data analysis project examining how social media usage patterns 
affect the mental health, sleep, and academic performance of 1,200 teenagers.

##  What This Project Explores

- Which platforms teenagers use most — and why that matters psychologically
- How daily social media hours relate to sleep, stress, and academic performance
- Patterns in anxiety, addiction, and depression indicators across age and gender
- What the data actually says vs. what people assume about social media and teens

##  Key Findings

- Average teen spends **4.5 hours/day** on social media — more than any single subject at school
- Average sleep is **6.4 hours** — below every medical recommendation for adolescents
- **Instagram** is the dominant platform, linked to social comparison and anxiety
- Stress levels average **5.4/10** — chronically elevated, never truly at rest
- Only 31/1200 flagged for depression — but stress and anxiety preconditions exist far more broadly
